ARCameraContext.reset

解释：重置相机
Web 态说明：Web 态暂不支持 ARCameraContext 相关方法。在 Web 态调用该方法会做 打开百度 App 对应小程序页面 的降级处理。

# 方法参数

Object object

# object参数说明

参数 类型 默认值 必填 说明
success Function 否 接口调用成功的回调函数
fail Function 否 接口调用失败的回调函数
complete Function 否 接口调用结束的回调函数（调用成功、失败都会执行）
